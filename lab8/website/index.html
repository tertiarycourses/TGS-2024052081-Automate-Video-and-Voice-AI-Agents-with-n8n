Interactive Avatar

A HeyGen interactive avatar streams live video and talks back in real time — like a video call with an AI.

How to get the URL

  1. 1 Open HeyGen → Interactive Avatar (labs.heygen.com/interactive-avatar).
  2. 2 Pick an avatar, configure its knowledge base / LLM.
  3. 3 Click Share / Embed and copy the link.
  4. 4 Paste it above and Load — allow microphone when asked.
n8n option (advanced): instead of the share URL, mint a session token from the LiveAvatar API through an n8n webhook and drive the avatar with the Streaming SDK — see Lab 7 in the Learner Guide.
